# Ein Foto per WhatsApp verschicken

Diese Anleitung zeigt, wie Sie ein Foto aus Ihrer Foto-Galerie an eine Person in WhatsApp senden – zum Beispiel an Ihre Kinder oder Enkelkinder.

1. Öffnen Sie die **WhatsApp-App** auf Ihrem Handy (grünes Symbol mit weißem Sprechblasen-Telefon).
2. Tippen Sie oben auf die Suchleiste oder auf die Liste der Chats und wählen Sie die Person aus, der Sie das Foto schicken möchten. Wenn Sie noch keinen Chat mit dieser Person haben: tippen Sie auf das runde Stift- oder Plus-Symbol unten rechts, um einen neuen Chat zu starten.
3. Im Chat-Fenster sehen Sie unten ein kleines **Büroklammer-Symbol** oder ein **Plus-Symbol**. Tippen Sie darauf.
4. Es öffnet sich eine Auswahl. Tippen Sie auf **„Galerie"** oder **„Fotos"**.
5. Suchen Sie das gewünschte Foto in Ihrer Galerie und tippen Sie einmal darauf, damit es blau markiert wird (ausgewählt).
6. Tippen Sie unten rechts auf den **Pfeil** oder auf **„Senden"**.
7. Das Foto wird jetzt verschickt. Sie sehen es im Chat-Verlauf erscheinen.

**Tipp:** Wenn das Foto nicht in der Galerie erscheint, warten Sie einen Moment – manchmal dauert es kurz, bis ein neu aufgenommenes Foto gespeichert ist.

**Wenn etwas nicht klappt:** Manche Handys zeigen die Symbole an leicht anderen Stellen an (je nach Handy-Marke). Grundsätzlich gilt immer: Büroklammer oder Plus antippen → Galerie auswählen → Foto antippen → Senden-Pfeil drücken.
